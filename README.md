# Wasko AI Tutor

Offline AI-powered Digital Skills, Cybersecurity and Entrepreneurship Tutor for Africa.

## Problem

Millions of African learners lack reliable internet access and cannot benefit from cloud-based AI tools.

## Solution

Wasko AI Tutor provides:

- Cybersecurity training
- Linux tutorials
- Networking education
- Entrepreneurship guidance

All running completely offline on an 8GB laptop.

## Features

✅ Offline AI Chat
✅ Cybersecurity Learning
✅ Linux Administration Training
✅ Business Plan Generation
✅ Hausa Language Support
✅ Local Knowledge Base
